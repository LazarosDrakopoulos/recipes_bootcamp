# Recipe Finder App

This is one of my first projects, created during the bootcamp I attended in 2024.

## About the Project
The Recipe Finder App allows users to search for recipes using the Spoonacular API.  
Users can enter a recipe name, and the app fetches recipe details including ingredients and cooking instructions.

## Features
- Search recipes by name
- Fetch data from Spoonacular API
- Display ingredients list
- Show cooking instructions
- Practice working with APIs, JavaScript async/await, and DOM manipulation

## Technologies Used
- HTML
- CSS
- JavaScript

## What I Learned
Through this project, I improved my understanding of:
- API requests with `fetch()`
- Async/Await
- JSON data handling
- DOM manipulation
- Error handling

This project represents an important step in my programming journey, as it helped me build confidence with real-world web development concepts during my early learning phase.
